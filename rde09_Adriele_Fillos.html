Incorporando Multimídias ao site

Tag iframe

A tag <iframe> é usada para incorporar outro documento ao documento HTML.

Sintaxe:
<iframe src="URL" height="altura em pixels" width="largura em pixels" title="Título"></iframe>

Incorporar videos ao site

Tag vídeo

O elemento HTML <video> é utilizado para incorporar conteúdo de video em um documento HTML ou XHTML.

Sintaxe:
<video poster=""></video>

Tag de audio

O elemento audio é utilizado para embutir conteúdo de som em um documento HTML ou XHTML. O elemento audio foi adicionado como parte do HTML5.

Sintaxe:
<audio preload=""></audio>

Tag Picture

O elemento HTML picture é um container usado para especificar múltiplos elementos <source> para um elemento específico <img> contido nele.

Sintaxe:
<picture ></picture>
